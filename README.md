# Meadow.js
Add a flower garden to your website

# Usage

``` js
// Active Bird
Meadow.createBird()
// Active Garden
Meadow.createGardens()

// or active all
Meadow.all()
```

# Settings
Meadow.js requires you to specify where you store its resources.  The default is "images"

```
Meadow.path = "/images"
```
